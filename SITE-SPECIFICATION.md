## 14. Content Maintenance

All content is sourced from CSVs in `/cms/`. To update any text on the site:

1. Edit the relevant CSV (`text.csv`, `menu.csv`, `stats.csv`, `portfolios.csv`, `audit.csv`, `sectors.csv`, `locations.csv`)
2. Run `python cms/build.py`
3. Commit and push to GitHub
4. CI auto-minifies CSS/JS and deploys via GitHub Pages

**No HTML edits required for content changes.**
